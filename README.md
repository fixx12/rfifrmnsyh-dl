## ✨ Overview

**rfifrmnsyh-dl** adalah universal media downloader berbasis Node.js yang memungkinkan developer mendownload video, audio, dan foto dari berbagai platform sosial media **tanpa API pihak ketiga**.

### 🎯 Use Cases
- WhatsApp Bot
- Automation System
- Backend Tools
- CLI Downloader
- AI Automation

### 📱 Supported Platforms
- ✅ TikTok & TikTok Lite
- ✅ YouTube
- ✅ Facebook & Facebook Lite
- ✅ Pinterest
- ✅ SoundCloud
- ⚠️ Instagram (butuh cookie)
- ❌ Spotify (DRM)
