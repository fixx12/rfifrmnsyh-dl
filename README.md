## ✨ Overview

**rfifrmnsyh-dl** adalah universal media downloader berbasis Node.js yang memungkinkan developer mendownload video, audio, dan foto dari berbagai platform sosial media.

Library ini dibuat untuk:

- WhatsApp Bot
- Automation System
- Backend Tools
- CLI Downloader
- AI Automation

Dirancang agar bisa berjalan di:

- Node.js
- Linux
- VPS
- Termux Android

**Tanpa bergantung pada API downloader pihak ketiga.**
